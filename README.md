# Angular Demo

This project is generated with [yo angular generator](https://github.com/yeoman/generator-angular)
version 0.15.1.

##项目安装
`sudo npm install -g generator-angular`  
`sudo npm install -g yo`  
`sudo npm install -g gulp-cli`  
`yo angular`  


## Build & development

`gulp start:client`

## Testing

`gulp start:server:test`
